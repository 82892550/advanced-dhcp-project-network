# Advanced DHCP Network Configuration

## Project Overview

This project demonstrates an advanced DHCP configuration using Cisco Packet Tracer.
The network consists of multiple switches, end devices, and a router connecting two different networks.

The DHCP service automatically assigns IP addresses to clients in different subnets.

## Network Topology

The network contains two main networks:

* Network 1: 13.0.0.0
* Network 2: 12.0.0.0

A router is used to connect both networks and allow communication between them.

## Features Implemented

* Dynamic IP assignment using DHCP
* Multiple DHCP pools
* Excluded IP addresses
* Inter-network communication through router
* Multiple switches connected in a hierarchical topology

## Devices Used

* 1 Router
* Multiple Switches
* PCs
* Servers

## Technologies

* DHCP
* IPv4 Addressing
* Routing
* Switching

## Testing

* Clients receive IP addresses automatically.
* Successful connectivity between devices using ping.
